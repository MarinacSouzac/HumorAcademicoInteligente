# HumorAcademicoInteligente
Aplicação completa (API + Front-end) que entrega recomendações para estudantes baseadas no humor atual. Desenvolvida com Node.js, Express, MongoDB e front-end em HTML/CSS/JS puro, consumindo todas as rotas da API e exibindo tudo em um layout estilo Pinterest. Inclui CRUD completo, filtro, estatísticas e interface intuitiva.
